# Email & SMS Notifications

**DeadEgos now uses Twilio for email (SendGrid) and SMS.**

See **[TWILIO_NOTIFICATIONS.md](./TWILIO_NOTIFICATIONS.md)** for the full setup guide.

Quick env vars:

```env
SENDGRID_API_KEY=SG...
SENDGRID_FROM_EMAIL=DeadEgos <orders@deadegos.co>
TWILIO_ACCOUNT_SID=AC...
TWILIO_AUTH_TOKEN=...
TWILIO_PHONE_NUMBER=+1...
```
